Real predictions from real data. A gamechanger in the industry.

The predictive tool was created via a neural network to identify positive and negative polyp cases. The data was derived from the Kvasir dataset, which was collectd using endoscopic equipment at the Vsetre Viken Health Trust in Norway. The dataset consists of 1,000 images of polyp and non-polyp cases, which was used to build a neural network using the Keras library with Python.
